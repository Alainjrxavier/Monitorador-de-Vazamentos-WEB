O que você vai precisar:
1 - No arquivo key.env, defina as APIS do flask e do HaveIBeenPwned
1.1 - A API do flask você mesmo pode definir.
1.2 - a API do HaveIbeenPwned pode ser adquirida aqui: https://haveibeenpwned.com/Subscription
2 - Pronto, após selecionar a pasta com o código no seu Visual Studio ou qualquer outro programa de código, é só executar um único comando:
pip install -r requirements.txt
agora o código está totalmente funcional!
